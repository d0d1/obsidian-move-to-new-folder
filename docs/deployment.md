# Deployment

Use Obsidian's official plugin documentation for development, local installation, and submission:

- Build a plugin: https://docs.obsidian.md/Plugins/Getting%20started/Build%20a%20plugin
- Releasing and submission requirements: https://docs.obsidian.md/Plugins/Releasing/Submission%20requirements%20for%20plugins
- Submit your plugin: https://docs.obsidian.md/Plugins/Releasing/Submit%20your%20plugin

This project does not define a separate deployment workflow beyond Obsidian's standard plugin process.
